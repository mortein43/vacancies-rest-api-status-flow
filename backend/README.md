# Vacancies REST API

## Features

- Full CRUD
- Status transitions (Draft → Open → Paused → Closed)
- Validation with class-validator

## Endpoints

- GET /vacancies
- GET /vacancies/:id
- POST /vacancies
- PATCH /vacancies/:id
- PATCH /vacancies/:id/status
- DELETE /vacancies/:id

## Run

npm install
npm run start
